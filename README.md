# News

A Hacker News reader Android app built with OkHttp, Gson, and Glide.

## Features

- Fetch top stories from Hacker News API
- Parallel network requests for better performance
- Random sampling for diverse content display

## Tech Stack

- OkHttp 4.12.0 — Network requests
- Gson 2.11.0 — JSON parsing
- Glide 4.16.0 — Image loading
- RecyclerView — List display

## Build

```bash
./gradlew assembleDebug
```
